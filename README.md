ğŸ“Œ What is a File System?
A file system is the method and data structure an operating system uses to control how data is stored, retrieved,
and organized on storage devices like hard drives, SSDs, or flash drives.

In short, a file system:
-> Organizes data into files and directories
-> Manages read/write access
-> Keeps track of metadata (file name, size, permissions, etc.)
-> Ensures data consistency and integrity
